NGINX

$> apt install nginx

■ Move to /var/www/, make a dir with your site name to save our html files

■ Now, move to /etc/nginx/ to configure our server

■ Inside sites-available, we'll create a file with your site name, and we'll configure our server

■ Inside this file, write the next code

server {
    listen 80;
    root /var/www/site_name;
    index index.html;
    server_name site_name www.site_name;
    location / {
        try_files $uri $uri/ =404;
    }
}

■ Now make a symbolic link in sites-enabled to make nginx know that it needs to run it

    $> ln -s /etc/nginx/sites-available/site-name /etc/nginx/sites-enabled/site-name

■ Restart your nginx service and you can access to your site on localhost:80

    $> service nginx restart

SSH

$> apt install openssh-server

■ Move to /etc/ssh/ and edit sshd_config

■ Now we look for port, uncomment and add 4242

    Port 4242

■ Now look for PermitRootLogin and change it to yes

    PermitRootLogin yes

■ Now restart your ssh service and we're done

    $> service ssh restart

TOR

$> apt install tor

■ Move to /etc/tor/ and edit torrc file

■ Look for HiddenServiceDir and HiddenServicePort and uncomment it

    HiddenServiceDir /var/lib/tor/hidden_service/
    HiddenServicePort 80 127.0.0.1:80

■ Restart your tor service

    $> service tor restart

■ Now move to /var/lib/tor/hidden_service and cat Hostname and we have our .onion ready
